# yeesoontuck-site

Personal portfolio site — plain HTML/CSS with a little vanilla JS. No frameworks, no build step.

Live at: https://yeesoontuck.github.io/yeesoontuck-site

## Structure

```
public/
  index.html   # main page
  style.css    # styles
  main.js      # interactions
```

## Deploy

Push to `main`. GitHub Actions publishes the `public/` folder to GitHub Pages automatically.
